Frontend (Interfaz de usuario)

- HTML, CSS y Bootstrap → construcción de páginas responsivas y accesibles.


- JavaScript → interactividad y dinamismo en la aplicación web.

Backend (logica y procesamiento )


- PHP → desarrollo de la lógica de negocio y conexión con la base de datos.


- PHPMyAdmin / MySQL → gestión y administración de datos (usuarios, dispositivos, trazabilidad).

Servisor en la nube

- AWS (Amazon Web Services) → despliegue de la plataforma, escalabilidad y seguridad.

Control de versiones y colaboracion


- GitHub → repositorio central para subir actualizaciones, gestionar versiones y trabajar en equipo.

Inteligencia artificial y Desarrollo

- Copilot y Cursor AI → asistentes de programación que sugieren código y optimizan funciones.


- Visual Studio Code → entorno principal de desarrollo y depuración.

Despliegue y Acceso Remoto a AWS


FileZilla → transferencia segura de archivos al servidor (FTP/SFTP).


- PuTTY → acceso remoto y administración de servidores mediante SSH.
